Overlain code overlays

Flash-to-SRAM code overlays on an STM32G031K8 for deterministic performance experiments.

Overlain keeps bare-metal firmware small while running heavyweight crypto and benchmark routines at SRAM speeds. Custom linker sections map overlayable code segments, which are relocated right before execution.

  • Benchmarks RSA-2048 verify and Miller-Rabin tests with and without overlays.
  • Shows 1.34x speedup from executing latency-sensitive code from SRAM.
  • Documents overlay loader, linker script, and measurement workflow for reuse.
