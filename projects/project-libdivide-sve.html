ARM SVE

libdivide ARM SVE

ARM SVE support for libdivide's vector division paths.

Implemented SVE vector division paths for all widths with tests and documentation. Benchmarked on Graviton4 to compare against the existing NEON implementation.

  • Around 5x faster 16-bit division vs NEON from wider scalable vectors and native lane operations.
  • Around 3x faster 64-bit division vs NEON from SVE native vector multiply-high instructions.

Pull requests

  • PR #158: Added the initial ARM SVE backend for 16/32/64-bit signed and unsigned vector division, wired into CMake, tests, benchmarks, and documentation.
  • PR #159: Optimized the SVE backend and added an optional SVE2 path using predicated merge-adds and SVE2 halving-add instructions where they improved Graviton4 performance.
