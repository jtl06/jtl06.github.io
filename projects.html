Firmware & Tooling

BLT latency measurement platform

STM32 input-to-photon latency measurement device for benchmarking gaming PC responsiveness. Uses timers, circular DMA, and ISRs for microsecond-level resolution.

C++ & SIMD

SwiftFIX — SIMD FIX pre-parser

C++20 FIX tag-value pre-parser with single-pass structural scanning, header indexing, body-length validation, and AVX2 frame splitting.

FPGA & PCIe

PNVMe — FPGA NVMe device

Built a custom NVMe SSD controller on an Artix-7 FPGA that enumerates as a PCIe NVMe Linux block device. Substantiated the NVMe BAR and wrote a custom XDMA-to-NVMe harness backed by onboard DDR3.

Software & AI

Nycticorax — Discord bot

Built a Discord bot using Claude Code and Codex for iterative implementation and debugging. Features selective memory, hybrid retrieval, and Postgres storage while controlling latency and cost. Modularized the codebase so different AI tools and multi-agent routines could contribute coherently.
